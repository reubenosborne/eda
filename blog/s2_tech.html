Everything is a block

A guide to CSS positioning

Everything on the world wide web is a block. These blocks stack up on each other in a big column.

By default these blocks have a 'Static' position. If you put something on a web page you don't actually need to set the position yourself, by default, it will be static. But this can be changed.

There's another positioning value that we can call on called 'Absolute'. It has similarities to Fixed but instead positions the element anywhere on the page we want. That's right, anywhere.

Aha! Right over the login form.

The 'absolute' positioning is dependant on any parent element that has 'Absolute' or 'Relative' positioning. What does this mean? It means that if the logo was inside another div that had a 'Relative' or 'Absolute' positioning it would be positioned relatively to that element. Because it doesn't, it gets positioned relative to the html document.
